# Ideas futuras

## Neon Control

Ecofandy Control podrá evolucionar en el futuro a Neon Control, una plataforma SaaS para otros fabricantes de letreros.

La idea es que cada empresa pueda personalizar:

- Logo
- Nombre comercial
- Colores
- Dirección
- Contacto
- Redes sociales
- Usuarios
- Flujos de trabajo

Ecofandy Neón será el primer caso real de uso.

Primero se construirá y probará Ecofandy Control internamente. Después, cuando sea estable, se evaluará convertirlo en Neon Control con arquitectura multiempresa.
